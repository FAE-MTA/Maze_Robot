Bài 5. Điều khiển động cơ RC Servo

Bài học giúp học viên hiểu nguyên lý hoạt động của servo và cách điều khiển góc quay bằng Arduino để ứng dụng trong robot.

1. Mục tiêu bài học

  • Hiểu nguyên lý hoạt động của RC Servo
  • Biết cách đấu nối servo với Arduino
  • Điều khiển góc quay từ 0° đến 180°
  • Ứng dụng servo trong robot

2. Giới thiệu RC Servo

RC Servo là loại động cơ đặc biệt có thể điều khiển chính xác góc quay. Servo thường được sử dụng trong robot để điều khiển cơ cấu như gạt, đẩy hoặc quay cảm biến.

Hình 2.1. Động cơ RC Servo thường dùng trong robot.

2.1. Cấu tạo dây servo

  • Đỏ: VCC (5V)
  • Nâu/Đen: GND
  • Vàng/Cam: Tín hiệu điều khiển
Servo nhận tín hiệu PWM từ Arduino để điều khiển góc quay.

3. Đấu nối Servo với Arduino

  • Dây đỏ → 5V
  • Dây đen → GND
  • Dây vàng → chân D15
Hình 3.1. Sơ đồ kết nối servo với Arduino Mega 2560.

Không cấp nguồn servo từ chân Arduino nếu servo lớn → nên dùng nguồn ngoài.

4. Thư viện điều khiển Servo

Arduino cung cấp thư viện Servo.h để điều khiển servo dễ dàng. 

#include <Servo.h>
servo.attach(15);   // Gắn servo vào chân D15
servo.write(90);    // Quay đến góc 90 độ

5. Code điều khiển servo cơ bản


#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(15);   // Gắn servo vào chân D15
}

void loop() {

  myServo.write(0);     // Quay về 0 độ
  delay(1000);

  myServo.write(90);    // Quay 90 độ
  delay(1000);

  myServo.write(180);   // Quay 180 độ
  delay(1000);
}

6. Điều khiển servo bằng nút nhấn

Kết hợp kiến thức bài trước, ta có thể dùng nút nhấn để điều khiển góc quay của servo. 


#include <Servo.h>
#define BUTTON_PIN A0   // Đổi sang A0
Servo myServo;
int angle = 0;

void setup() {
  myServo.attach(15);   // Servo chân D15
  Serial.begin(9600);   // Debug ngưỡng
}
void loop() {
  int value = analogRead(BUTTON_PIN); // Đọc A0
  Serial.println(value); // In ra để kiểm tra
  // ===== NHẤN NÚT =====
  if (value > 100) {   // Ngưỡng nút (tùy chỉnh theo mạch)
    angle += 45;       // Tăng góc
    if (angle > 180) angle = 0;
    myServo.write(angle);
    delay(300);        // chống dội
  }
}

7. Ứng dụng trong robot

  • Gạt vật thể trong sân thi đấu
  • Quay cảm biến siêu âm
  • Điều khiển cơ cấu nâng/hạ
Servo thường dùng cho chuyển động chính xác, không dùng để di chuyển robot.

8. Bài tập thực hành

  • Viết chương trình quay servo qua 3 góc 0° – 90° – 180°
  • Dùng nút nhấn để điều khiển servo
  • Viết chương trình quay servo liên tục qua lại

9. Lỗi thường gặp

Servo rung → nguồn không đủ hoặc dây nối lỏng
Servo không quay → sai chân hoặc chưa attach()
Servo quay sai góc → cấp tín hiệu PWM không đúng
