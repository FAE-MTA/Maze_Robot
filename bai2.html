Bài 2. Điều khiển LED

Bài học giúp học viên làm quen với điều khiển thiết bị đầu ra thông qua LED vòng WS2812, dùng để hiển thị trạng thái hoạt động của robot.

1. Mục tiêu bài học

  • Hiểu khái niệm OUTPUT
  • Biết điều khiển LED vòng WS2812
  • Hiểu cách hiển thị màu RGB
  • Sử dụng LED để báo trạng thái robot

2. Lý thuyết

2.1. LED thường

LED thường chỉ có 2 trạng thái: bật hoặc tắt. Arduino điều khiển bằng HIGH (bật) và LOW (tắt).

2.2. LED WS2812 là gì?

WS2812 là LED RGB thông minh, mỗi LED có thể hiển thị màu riêng. Chỉ cần 1 chân tín hiệu để điều khiển toàn bộ dãy LED.

2.3. Màu RGB

  • Đỏ: (255, 0, 0)
  • Xanh lá: (0, 255, 0)
  • Xanh dương: (0, 0, 255)
  • Trắng: (255, 255, 255)
Mỗi màu là tổ hợp của 3 thành phần: Red - Green - Blue

3. Sơ đồ kết nối chân

LED vòng WS2812 được kết nối với Arduino Mega như sau:

  • DIN → D13
  • VCC → 5V
  • GND → GND
Lưu ý:
  • Phải nối chung GND giữa Arduino và LED
  • Cấp nguồn đủ 5V để LED hoạt động ổn định
  • Nếu sử dụng nhiều LED, Nối chân DIN với chân DOUT của LED trước

4. Nguyên lý hoạt động

Arduino gửi dữ liệu màu đến LED thông qua chân DATA. Mỗi LED nhận dữ liệu và hiển thị màu tương ứng.

Luồng hoạt động: Arduino → DATA → LED WS2812 → Hiển thị

5. Code ví dụ


#include <Adafruit_NeoPixel.h>

#define LED_PIN 13
#define NUM_LEDS 16

Adafruit_NeoPixel ring(NUM_LEDS, LED_PIN, NEO_GRB + NEO_KHZ800);

void setup() {
  ring.begin();
  ring.show();
}

void loop() {

  // bật toàn bộ LED màu đỏ
  for (int i = 0; i < NUM_LEDS; i++) {
    ring.setPixelColor(i, ring.Color(255, 0, 0));
  }

  ring.show();

}


#include <Adafruit_NeoPixel.h>

#define LED_PIN 13
#define NUM_LEDS 16

Adafruit_NeoPixel ring(NUM_LEDS, LED_PIN, NEO_GRB + NEO_KHZ800);

int speedDelay = 40;
int colorShift = 0;

// ===== Tạo màu rainbow =====
uint32_t wheel(byte pos) {
  pos = 255 - pos;

  if (pos < 85) {
    return ring.Color(255 - pos * 3, 0, pos * 3);
  }
  if (pos < 170) {
    pos -= 85;
    return ring.Color(0, pos * 3, 255 - pos * 3);
  }
  pos -= 170;
  return ring.Color(pos * 3, 255 - pos * 3, 0);
}

// ===== Fade đuôi =====
void fadeAll(byte amount) {
  for (int i = 0; i < NUM_LEDS; i++) {
    uint32_t c = ring.getPixelColor(i);

    uint8_t r = (c >> 16) & 0xFF;
    uint8_t g = (c >> 8) & 0xFF;
    uint8_t b = c & 0xFF;

    r = (r * amount) / 255;
    g = (g * amount) / 255;
    b = (b * amount) / 255;

    ring.setPixelColor(i, r, g, b);
  }
}

void setup() {
  ring.begin();
  ring.show();
}

void loop() {

  for (int i = 0; i < NUM_LEDS; i++) {

    fadeAll(180);

    int colorIndex = (i * 16 + colorShift) % 256;

    ring.setPixelColor(i, wheel(colorIndex));

    ring.show();
    delay(speedDelay);
  }

  colorShift++;
}
Hiệu ứng: LED chạy vòng + đổi màu.

6. Bài tập thực hành

  • Làm LED chạy từng bóng một
  • Tạo hiệu ứng LED quay vòng
  • Tạo các trạng thái Robot với quy ước màu:
    • Xanh lá → sẵn sàng
    • Đỏ → lỗi
    • Xanh dương → đang chạy
